E-Commerce Website

My e-commerce website was built using asp.net core and follows the MVC design pattern, with Dependency Injection and EF Core for seamless integration and optimal performance. I implemented the Repository pattern to manage data access, and integrated a PayPal payment gateway to provide secure and reliable payment processing. My website is highly secured and optimized, providing a smooth browsing experience for all users.

Main Page On Desktop, Tablet and Phone

                 I have placed a great emphasis on designing a high-quality database. To achieve this, I have ensured that the database is normalized, meaning that each table contains only one type of data and that data is not duplicated across multiple tables. This ensures that the data is consistent and reduces redundancy.

Products On Desktop, Tablet and Phone

                 I have implemented the Repository Pattern to achieve a better separation of concerns and improve the overall design of the application. The repository pattern provides a layer of abstraction between the data access layer and the business logic layer, which allows for easier maintenance and testing of the application.

Cart Page On Desktop, Tablet and Phone
